# Adaptive Acoustic Emission Condition Monitoring System

## Overview

This project investigates FPGA-accelerated acoustic emission monitoring for early detection of tribological degradation and bearing faults in rotating machinery.

The system combines:

- Acoustic Emission Sensing
- Embedded Signal Processing
- Adaptive Thresholding
- STM32-Based Edge Processing
- FPGA Acceleration

The project is based on a patented adaptive-threshold methodology for predictive maintenance applications.

---

## Research Motivation

Unexpected bearing failures cause significant downtime and maintenance costs in industrial machinery.

Conventional vibration monitoring techniques often detect faults after substantial degradation has already occurred.

Acoustic emission sensing can detect microscopic fault activity significantly earlier, enabling predictive maintenance strategies.
